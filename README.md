# 📊 Job Tracker (Frontend)

A modern React-based Job Tracking application that helps users manage and monitor their job applications in one place. Built with a clean UI, dark mode support, and local storage persistence.

---

## 🚀 Features

- ➕ Add job applications (Title + Company)
- 📌 Track application status:
  - Applied
  - Interview
  - Rejected
  - Accepted
- ✏️ Update job status dynamically
- ❌ Delete job entries
- 📊 Real-time statistics dashboard
- 🌙 Dark / Light mode toggle
- 💾 Persistent storage using localStorage
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- React (useState, useEffect)
- JavaScript (ES6+)
- HTML5 & CSS3
- LocalStorage API

---


---

## ⚙️ How It Works

1. User adds a job (title + company)
2. Job is stored in state and saved to `localStorage`
3. Dashboard updates automatically with stats
4. User can:
   - change status
   - delete job
5. Theme preference is saved and applied on reload

---

## 🌙 Dark Mode

The app supports dark mode with persistent theme storage.  
User preference is saved in `localStorage` and applied on reload.
