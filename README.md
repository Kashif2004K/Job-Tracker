# 📊 Job Tracker (Frontend)

A modern React-based Job Tracking application that helps users manage and monitor their job applications in one place. Built with a clean UI, dark mode support, and local storage persistence.

---

## 🚀 Features

- ➕ Add job applications (Title + Company)
- 📌 Track application status:
  - Applied
  - Interview
  - Rejected
  - Accepted
- ✏️ Update job status dynamically
- ❌ Delete job entries
- 📊 Real-time statistics dashboard
- 🌙 Dark / Light mode toggle
- 💾 Persistent storage using localStorage
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- React (useState, useEffect)
- JavaScript (ES6+)
- HTML5 & CSS3
- LocalStorage API
